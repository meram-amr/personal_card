# Personal Card — Developer Profile Card

A personal profile card built with HTML and Tailwind CSS, focusing on clean design and responsive layout.

## Live Demo

[View Live](https://meram-amr.github.io/personal_card/)

---

## Features

* Personal information display (name, role, links)
* Built using Tailwind CSS utility-first approach
* Responsive and mobile-friendly layout
* Lightweight implementation using HTML and CSS

---

## Tech Stack

| Technology   | Purpose               |
| ------------ | --------------------- |
| HTML5        | Structure             |
| Tailwind CSS | Utility-first styling |
